# Ease-Engine

**Ease Engine** is a 2D Game/Rendering Engine


## Installation
- See [Dependencies](#-Dependencies)
- ``` make ``` to build the engine
- ``` make run ``` to run.  output is located at build/linux/a.out

## Dependencies
- There is none
